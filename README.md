## Установка проекта
Склонируйте репозиторий:

```bash
git clone https://github.com/yato90/countries-app.git
cd countries-app
```

Установка зависимостей:

```bash
npm install
```

Запуск:

```bash
npm start
```

Доступен по ссылке: http://localhost:3000/countries-app
